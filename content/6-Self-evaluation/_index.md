## Internship Self-Assessment

During my 12-week internship (05/01/2026 - 18/04/2026), I had the opportunity to develop my Frontend Development skills while working in an AWS Cloud environment. Starting from basic React knowledge, I was able to contribute to UI development and successfully integrate the frontend with a backend system deployed on AWS.

### Project Overview

During the internship, I worked on the frontend of a web system deployed on AWS Cloud. My main responsibilities were building user interfaces, integrating APIs, and ensuring the frontend worked reliably when connected to the backend infrastructure.

**Technical Achievements:**

- Developed the frontend with **ReactJS** and integrated it with backend APIs hosted on **Amazon EC2**
- Connected the frontend to the backend system via REST APIs
- Understood how the frontend works with the database **Amazon RDS** through backend APIs
- Used **Amazon S3** to display or manage static files (if applicable)
- Understood the system architecture using:
  - **VPC** for network infrastructure management
  - **IAM** for controlling access to AWS services
  - **API Gateway** for managing API endpoints (if used)
  - **Route 53** for domain management and routing
- Configured API URLs and environment variables to match deployment environments

**Development Skills:**

- Built UI in **ReactJS** using a component-based architecture
- Called APIs with Axios/Fetch and handled returned data
- Worked with authentication flows between frontend and backend
- Debugged common issues:
  - CORS errors when calling APIs
  - Network errors after deployment
  - Incorrect backend endpoint or port
- Improved UI and user experience

**Soft Skills:**

- Collaborated with backend developers to test APIs
- Proactively studied AWS documentation related to the system
- Improved my ability to read and understand technical documentation
- Wrote worklog reports and documented issues encountered during development
