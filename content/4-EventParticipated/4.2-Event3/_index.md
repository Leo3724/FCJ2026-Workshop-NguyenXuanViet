# "Cloud Mastery 2026 #3 AWS Security & Networking"

### Event Objectives

- Provide an overview of IAM, Networking, and system security on AWS.
- Build a foundation in access management, network design, and application protection.
- Connect theory with practice through demos and concrete examples.

### Speaker Details & Presentation Topics

**1. Huynh Hoang Long & Dang Thi Minh Thu - FCAJ Cloud Engineer Ambassador**
* **Topic:** IAM on AWS
* **Key content:** Focused on IAM concepts and how to manage identity and access permissions in AWS systems. The session covered IAM best practices, SSO mechanisms, how to apply SCP and Permission Boundaries for advanced access control, and security techniques such as credential rotation and Access Analyzer. The presentation included a practical demo.

**2. Lam An Thinh & Nguyen Phan Quoc Viet - Security Engineer Intern**
* **Topic:** Networking on AWS
* **Key content:** Presented an overview of networking in AWS, including VPC and CIDR design, subnet segmentation, Internet Gateway (IGW) and Route Table configuration, and using VPC Endpoints to optimize internal connectivity.

**3. Lam Tuan Kiet - DevOps**
* **Topic:** AWS Network & Application Protection
* **Key content:** Introduced AWS security services such as AWS WAF (Web Application Firewall), AWS Shield, AWS Network Firewall, and AWS Firewall Manager, with a focus on protecting systems and applications from threats.

### Key Highlights

#### 1. IAM and access management
- Understand what IAM is and its role in AWS systems.
- Apply best practices to manage access permissions securely.
- Learn the SSO mechanism in enterprise environments.
- Use SCP and Permission Boundaries for advanced permission control.
- Apply credential rotation and Access Analyzer to strengthen security.
- Observe a practical demo.

#### 2. Networking on AWS
- Understand how networking works in AWS.
- Design VPC and use CIDR appropriately.
- Segment subnets for system architecture.
- Configure IGW and Route Tables for traffic routing.
- Use VPC Endpoints to optimize internal connections.

#### 3. Network and application security on AWS
- Learn how AWS WAF protects web applications.
- Understand the role of AWS Shield in DDoS protection.
- Apply AWS Network Firewall to control network traffic.
- Use AWS Firewall Manager to centrally manage security policies.
