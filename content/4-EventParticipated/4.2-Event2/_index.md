# "Cloud Mastery 2026 #2 DevOps"

### Event Objectives

- Provide an overview of Kubernetes, Infrastructure as Code (IaC), and Elixir.
- Provide a foundation for building, deploying, and operating systems on the cloud.
- Connect theoretical knowledge with practical implementation through demos.

### Speaker Details & Presentation Topics

**1. Bao Huynh - Junior Cloud Native Developer - Endava Vietnam**
* **Topic:** Kubernetes
* **Key content:** The Kubernetes session focused on solving container orchestration challenges, starting with an overview, architecture, and core system components. The presentation went deeper into important Kubernetes objects, how to work with manifests, and how to use `kubectl`. It also included a live demo, guidance on how to start learning Kubernetes, a structured learning path, and supporting tools such as Helm and K9S.

**2. Thinh Nguyen - DevOps Engineer**
* **Topic:** Infrastructure as Code with Terraform on AWS
* **Key content:** The presentation focused on the concept of Infrastructure as Code (IaC) and its role in modern infrastructure management. Content included an overview of common IaC tools, an introduction to AWS CloudFormation, AWS CDK, and Terraform, along with a live demo of real infrastructure deployment on AWS.

**3. Nguyen Ta Minh Triet - SAP Developer Intern, Bosch GSV**
* **Topic:** Introduction to Elixir
* **Key content:** Covered core Elixir fundamentals, concurrent programming, and the OTP mechanism for fault tolerance. The session also discussed applying Elixir in DevOps pipelines, with live demos and practical examples.

### Key Highlights

#### 1. Kubernetes in practice
- Analyze challenges in container orchestration.
- Understand Kubernetes architecture and core components.
- Work with Kubernetes objects, manifests, and `kubectl`.
- Observe a real deployment demo.
- Get guidance on how to start and learn Kubernetes effectively.
- Get familiar with supporting tools such as Helm and K9S.

#### 2. Infrastructure as Code (IaC) on AWS
- Understand what Infrastructure as Code is and its role in DevOps.
- Compare common IaC tools.
- Learn AWS CloudFormation, AWS CDK, and Terraform.
- Observe an infrastructure deployment demo using Terraform.

#### 3. Elixir and high-load systems
- Understand core Elixir fundamentals.
- Learn concurrent programming concepts in Elixir.
- Explore OTP and how to build fault-tolerant systems.
- Apply Elixir in DevOps pipelines.
- Observe demos and practical examples.
