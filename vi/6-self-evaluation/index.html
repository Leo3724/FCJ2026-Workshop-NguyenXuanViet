Tự đánh giá

Đánh giá Quá trình Thực tập

Trong 12 tuần thực tập (05/01/2026 - 18/04/2026), tôi đã có cơ hội phát triển kỹ năng Frontend Development kết hợp với việc làm việc trong môi trường Cloud AWS. Từ kiến thức React cơ bản, tôi đã có thể tham gia phát triển giao diện và tích hợp thành công frontend với hệ thống backend được triển khai trên AWS.

Tổng quan Dự án

Trong quá trình thực tập, tôi tham gia phát triển frontend cho một hệ thống web được triển khai trên AWS Cloud. Công việc chính của tôi là xây dựng giao diện người dùng, tích hợp API, và đảm bảo frontend hoạt động ổn định khi kết nối với backend infrastructure.

Thành tựu Kỹ thuật:

  • Phát triển frontend bằng ReactJS và tích hợp với backend API được host trên Amazon EC2
  • Kết nối frontend với hệ thống backend thông qua REST API
  • Hiểu cách frontend làm việc với hệ thống database Amazon RDS thông qua backend APIs
  • Sử dụng Amazon S3 để hiển thị hoặc quản lý static files (nếu có)
  • Hiểu kiến trúc hệ thống sử dụng:
    • VPC để quản lý network infrastructure
    • IAM để kiểm soát quyền truy cập dịch vụ AWS
    • API Gateway để quản lý API endpoints (nếu sử dụng)
    • Route 53 để quản lý domain và routing
  • Cấu hình API URL và environment variables để phù hợp với môi trường deployment

Kỹ năng Phát triển:

  • Sử dụng ReactJS để xây dựng UI theo component-based architecture
  • Gọi API bằng Axios/Fetch và xử lý dữ liệu trả về
  • Xử lý authentication flow giữa frontend và backend
  • Debug các lỗi phổ biến:
    • CORS errors khi gọi API
    • Network errors sau khi deploy
    • Sai endpoint hoặc port backend
  • Cải thiện UI và trải nghiệm người dùng

Soft Skills:

  • Làm việc nhóm và phối hợp với backend developer để test API
  • Chủ động tìm hiểu tài liệu AWS liên quan đến hệ thống đang sử dụng
  • Rèn luyện khả năng đọc hiểu tài liệu kỹ thuật
  • Viết báo cáo worklog và ghi lại các vấn đề đã gặp trong quá trình làm việc

Điểm Mạnh Đã Chứng minh

1. Frontend Development Skills

  • Xây dựng UI bằng React theo component structure
  • Tích hợp API với backend system
  • Hiểu luồng hoạt động của frontend trong cloud architecture

2. Problem Solving

  • Debug lỗi CORS khi frontend gọi backend API
  • Fix lỗi API integration
  • Xử lý lỗi phát sinh sau khi hệ thống deploy

3. Self Learning

  • Chủ động tìm hiểu cách frontend hoạt động trong hệ thống AWS
  • Đọc AWS documentation liên quan tới EC2, S3, và networking
  • Tìm hiểu cách debug lỗi khi frontend deploy

4. Work Delivery

  • Hoàn thành các chức năng UI theo yêu cầu
  • Fix bugs theo feedback từ team
  • Đảm bảo frontend hoạt động ổn định khi tích hợp backend

Kết luận

Quá trình thực tập giúp tôi hiểu rõ hơn cách một Frontend Developer làm việc trong một hệ thống thực tế được triển khai trên Cloud AWS.

Từ kiến thức React cơ bản, tôi đã có thể:

  • Tích hợp frontend với backend API chạy trên EC2
  • Hiểu cách frontend tương tác với hệ thống cloud
  • Debug các lỗi thực tế trong quá trình development và deployment
  • Làm việc theo quy trình của một software development team

Những kinh nghiệm này giúp tôi có nền tảng tốt để phát triển theo hướng Frontend Developer có hiểu biết về Cloud Architecture và tiếp tục nâng cao kỹ năng về React cũng như hệ thống cloud trong tương lai.
